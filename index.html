Swift UIKit
</Lumiere (iOS)>

Inspired by the OG Lumiere (Web), this is an extended version of the same app with more functionalities and an improved front-end design. This is an iOS app built with Swift UIKit that allows users to add, edit, and delete films they've watched or want to watch to the app's lists. The app features an authentication system using FirebaseAuth, a data management system with FirebaseFirestore, and a storage for user-uploaded images in FirebaseStorage. Two lists are implemented in the app, one storing the title, rating, and date of the movie that they've watched, while the other keeping score of the title of any unwatched films.

Many thanks to Lexie Shi for helping with the design of Lumiere's icon and the launch screen.

Now available for download on your iOS devices here.

Unity 3D
</Running Square>

My first complete Unity project inspired by a similar game I have seen earlier in the year. Completed all by myself with the occasional help to look up the implementation of certain UI or functionalities and reference to the Udemy Unity course by GameDev.tv. This is a 3D game involving a cube that is constantly accelerating across a game board to reach a destination. The difficulty of the game increases as the velocity of the cube increases, making it harder to control. There will be different types of obstacles on the way: spinning obstacles, arbitrarily falling obstacles, and stationary obstacles. This is a fun and minimalist game where you get to try your best to go to the finish line at 4,900m.

Many thanks to Lexie Shi for helping with the design of the level.

</Pac-Man 3D>

A 3D remake of the classic Pac-Man game with inspiration from Dark Deception. This game features the classic Pac-Man on his quest trying to eat as many gems as possible without being caught by the four ghosts that are controlled by Unity NavMesh Agents with the same algorithm as that of the classical Pac-Man (to learn more about the Pac-Man ghosts' movement algorithms, feel free to learn more </HERE>) This game also improves on the original approach of input control from only using the arrow keys to control the movement of Pac-Man to being able to control the movement of Pac-Man with both arrow keys and mouse.
